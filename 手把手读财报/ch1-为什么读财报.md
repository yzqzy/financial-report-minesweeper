## 第一节 财报是用来排除企业的

### 三个财报造假小故事

**故事一：扭亏为盈只需白纸两张**（SJXY公司）

公司经营亏损1亿元，通过三步操作实现"扭亏为盈"：
1. 将账面价值3000万的房产作价1.6亿抵还银行贷款
2. 再以1.6亿从银行买回同一套房产（房款暂欠）
3. 结果：获得1.3亿营业外收入 → 抵消1亿亏损 → 扭亏为盈

> [!important] 启示
> 操作过程并不违法。房产还是自己的，负债金额没变（只是从"贷款"变成"应付购房款"），但[[concepts/资产负债表|资产负债表]]上房产从3000万变成了1.6亿。**投资不能只看利润数据，还要了解利润的产生过程。**

**故事二：完美造假坑惨投资者**（DFDZ公司）

4年涨60多倍的大牛股，从上市第一天就在造假：
1. 改制时将贷款运作至公司体外，低价吃下1044万股
2. 上市后公司挪用近7亿资金买入自己公司股票
3. 财务部分别算出计划收入与实际收入的差额 → 销售部按需制作合同和发票
4. 券商部按缺口抛售个人账户股票，资金回流至空壳公司甲
5. 空壳公司甲与上市公司签订购货合同，形成"真实"交易
6. 累计炒作获利17亿元，其中约16亿变成公司主营收入

> [!warning] 识别要点
> 如果不了解公司在业内的地位和产品的市场份额，不懂得运用财报中的**[[concepts/毛利率|毛利率]]、净利率和税收数据**与行业内竞争对手对比，踩雷就是大概率结果。

**故事三：可任意调节的盈利**（XZJ公司，2014年被证监会处罚）

上市公司隐秘出资设立空壳公司甲 → 甲从上市公司采购 → 通过公司乙转手 → 卖给上市公司下属子公司丙。产品原地不动，只通过合同和仓单的转移完成利润操控：
- 需要盈利时：高价卖给甲，低价卖回给丙 → 上市公司大幅盈利
- 需要亏损时：反向操作

5年累计虚增收入6.6亿元，虚增利润2.2亿元。财报里的**毛利率、[[concepts/关联交易|关联交易]]和运输费用等数据**都可能成为破绽。

### 小故事，大智慧

三个故事的两个核心教训：
1. **只看公司利润数据，很容易被误导**
2. **财报中有很多细节，可以帮助我们排除造假企业，降低踩雷概率**

> [!tip] 芒格的智慧
> "反过来想，总是反过来想。" 如果我们能够不断排除竞争力低下、财报数据可疑的公司，坚持与具有核心竞争力、具备诚信管理层的公司为伍，想不赚钱反而会成为一件很困难的事情。

### 投资的核心：真实盈利能力 + 管理层诚信度

**真实盈利能力**：在自己擅长的主业领域，靠具有竞争力的产品或服务持续获取利润的能力。

以下企业是首先需要排除的对象：
- 靠卖资产、变卖家当获利的
- 靠股票或房产等投资对象的短期价格波动获利的
- 靠说不清道不明的[[concepts/关联交易|关联交易]]等手段创造收入的

## 第二节 财报阅读入门

### 关于财报的两个基础知识

#### 时间

- 部分国家和地区的企业可以自行设定经营年度起止日（如阿里巴巴的财年是4月1日至次年3月31日）
- **中国A股上市公司统一按自然年（1月1日至12月31日）为经营年度**
- 按季度披露：一季报、半年报（中报）、三季报和年报

| 报表类型 | 披露时限               | 特点                   |
| ---- | ------------------ | -------------------- |
| 年报   | 年度结束后4个月内          | **必须审计**，信息最详细，可信度最高 |
| 半年报  | 8月底之前              | 次详细                  |
| 一季报  | 一季度结束后1个月内（不得早于年报） | 相对简单                 |
| 三季报  | 10月底前              | 相对简单                 |

> [!note] 无法按时披露年报是警示信号
> 如果企业无法按时披露年报，会被交易所停牌处理。出现这种情况，若非是对该企业或该行业非常熟悉，建议远离。

#### 范围

无论是年报还是季报，都至少包括**三张报表**：[[concepts/资产负债表|资产负债表]]、[[concepts/利润表|利润表]]和[[concepts/现金流量表|现金流量表]]（俗称"三大表"）。

每张表又要求分别提供**母公司报表**和**[[concepts/合并报表|合并报表]]**：
- **母公司报表**：记录上市公司本部的资产负债及经营情况。对于所控制的子公司，只记录股权投资成本，以及子公司所做的现金分红
- **合并报表**：将上市公司本部和全部子公司视为一个"假想的企业集团"来制作。**投资者重点关注的就是合并报表数据**

### 三大报表阅读框架

#### [[books/手把手读财报/ch2-资产负债表|资产负债表]]

[[concepts/资产负债表|资产负债表]]是企业在报表日（即年末或季度末）结束那一瞬间，资产、负债及所有者权益的**定格照片**。详见[[books/手把手读财报/ch2-资产负债表|第二章]]。

```
资产负债表
────────────────────────────────────
资产（钱的去处）    │ 负债和股东权益（钱的来源）
                    │
流动资产            │ 流动负债
非流动资产          │ 非流动负债
                    │ ────────
                    │ 负债合计
                    │ 股东权益
────────────────────│────────────────
资产总计            │ 负债和股东权益总计
```

> [!important] 会计第一恒等式
> **资产 = 负债 + 股东权益**
> 
> 因为金钱不会无中生有，所以钱的来源和去处，总额永远是相等的。
> 
> 移项后：**股东权益（净资产）= 资产 - 负债**

#### [[books/手把手读财报/ch3-利润表|利润表]]

[[concepts/利润表|利润表]]也叫"损益表"，是一串加减法（详见[[books/手把手读财报/ch3-利润表|第三章]]）：

| 项目 | 公式 |
|------|------|
| A 营业总收入 | / |
| B 营业总成本 | / |
| C 非主营业务净收入 | / |
| D 营业利润 | A - B + C = D |
| E 营业外收支净额 | / |
| F 利润总额 | D + E = F |
| G 所得税 | / |
| H 净利润 | F - G = H |
| I 少数股东损益 | / |
| J 归属上市公司所有的净利润 | H - I = J |

**权责发生制：[[concepts/财务造假|财务造假]]的"温床"**

[[concepts/利润表|利润表]]依照"权责发生制"编制 — 以权利或责任的发生来确定"应该得到的收入"和"应该支付的费用"，不考虑实际是否收到或支出了现金。

**备受追捧的归母净利润**

- 合并利润表的净利润 - 少数股东损益 = **归属于母公司所有者净利润**（简称"归母净利润"）
- 归母净资产和归母净利润分别除以公司总股本 = **每股净资产**和**[[concepts/EPS|每股收益]]**
- **市净率 PB** = 股票价格 / 每股净资产 = 公司总市值 / 归母净资产
- **[[concepts/市盈率|市盈率]] PE** = 股票价格 / [[concepts/EPS|每股收益]] = 公司总市值 / 归母净利润

#### [[books/手把手读财报/ch4-现金流量表|现金流量表]]

[[concepts/现金流量表|现金流量表]]的原理比较简单，就和我们日常记账差不多（详见[[books/手把手读财报/ch4-现金流量表|第四章]]） — 年初有多少现金，共收到多少，付出去多少，还剩下多少。

三大活动分类：

| 活动类型 | 内容 | 说明 |
|---------|------|------|
| **经营活动** | 销售商品或提供劳务带来的现金收入及对应的现金支出 | 与企业投资和筹资活动无关的现金流入和流出，都归入经营活动 |
| **投资活动** | 固定资产、无形资产等长期资产的买入或处置，对外股权投资和处置，金融理财类投资等 | 现金及现金等价物范围内的形态转换不计算 |
| **筹资活动** | 发行股票、回购股票、分配股利、借债、还债、支付利息等 | 不包括企业日常经营活动中产生的应收应付款项 |

## 第三节 读财报前的准备工作

### 获得上市公司财报

三大官方下载渠道：
- **巨潮资讯网**：cninfo.com.cn（可查询深沪两地上市公司财报及公告）
- **上海证券交易所**：sse.com.cn
- **深圳证券交易所**：szse.cn

### 为什么选贵州茅台做范本？六大理由

1. **主营业务单一**：就是产酒卖酒，好理解
2. **公司组成相对简单**：[[concepts/合并报表|合并报表]]范围内子公司仅五家，其中一家占营业收入95%以上
3. **产销有可靠对应关系**：每年生产量与四五年后的销售量有基本可靠的对应关系
4. **产品畅销，先收钱后发货**：几乎不会有死账、坏账。[[concepts/存货|存货]]入账成本远低于出厂价（约7%），几乎不可能产生存货贬值
5. **市场研究资料丰富**：新手在阅读过程中遇到不懂的内容，很容易在网络上找到相关资料
6. **投资参照物价值高**：在中国股市做投资，几乎不可能绕开贵州茅台

## 第四节 重要提示

打开贵州茅台2013年财报，首先看到的是《重要提示》。

### 审计意见（最重要！）

> [!warning] 拿到财报第一步
> **先找到"标准无保留意见"这句话。没有这个意见的报告，直接丢进垃圾桶就好。**

审计意见可能出具五种，降格理解其真实意思：

| 会计师出具意见 | 真实意思对照 |
|-------------|-----------|
| 标准无保留意见的审计报告 | 造假迹象未被本人发现 |
| 附带说明的无保留意见审计报告 | 黑锅有人背，本人大胆收钱 |
| 保留意见审计报告 | 假报表，别看了 |
| 无法出具意见审计报告 | 本人拒绝和拙劣的骗子合作 |
| 否定意见审计报告 | 本人举报诈骗犯 |

### 四个重要概念

**1. 净利润**：看似简单，实则复杂。甲公司花70万买原材料 + 30万制造费 + 40万税费 → 200万卖给乙公司但只拿到欠条。会计记录净利润45万，但公司一毛钱现金没进账。

**2. 盈余公积金**：法律规定公司盈利后，必须留存至少10%作为盈余公积。累计超过注册资金的50%后，由董事会自行决定。

**3. 未分配利润**：历年净利润总和 - 盈余公积 - 分红。**未分配利润不等于现金**。

**4. 送股与转增的区别**：送股用利润/盈余公积，需缴税；转增用资本公积，不用缴税。

## 第五节 财报的结构

**重点阅读（必看）— 3节**：
1. **财务会计报告**（三张表 + 附注 = 重中之重）
2. **董事会报告**
3. **重要事项**

**次要内容 — 2节**：
1. **股份变动及股东情况**
2. **董事、监事、高级管理人员和员工情况**

## 关键概念速查

| 概念 | 定义 | 备注 |
|------|------|------|
| 会计第一恒等式 | 资产 = 负债 + 股东权益 | 金钱不会无中生有 |
| 净资产（股东权益） | 资产 - 负债 | 又叫股东权益 |
| 权责发生制 | 以权利或责任的发生来确认收入和费用 | 不考虑现金是否实际收支，造假的温床 |
| 归母净利润 | 合并净利润 - 少数股东损益 | 投资者最关注的利润指标 |
| [[concepts/EPS|每股收益]] | 归母净利润 / 总股本 | |
| [[concepts/市盈率|市盈率]] PE | 股价 / 每股收益 = 总市值 / 归母净利润 | Price/Earning |

> [!summary] 第一章核心 Takeaways
> 1. **财报是用来排除企业的** — 先排雷，再谈投资（→ [[books/手把手读财报/ch8-欺诈与反欺诈|第八章 排雷实操]]）
> 2. **拿到财报第一步** — 找"标准无保留意见"，没有的直接扔掉
> 3. **高手先看[[concepts/资产负债表|资产负债表]]** — 不是[[concepts/利润表|利润表]]
> 4. **[[concepts/利润表|利润表]]最容易造假** — 因为权责发生制，必须结合[[concepts/现金流量表|现金流量表]]验证
> 5. **三张表要交叉阅读** — 利润表看"好不好看"，现金流量表看"靠不靠谱"，资产负债表看"底子厚不厚"（→ [[books/手把手读财报/ch5-综合分析|第五章 综合分析]]）
> 6. **未分配利润 ≠ 现金** — 想知道公司有多少钱，看"货币资金"
> 7. **附注是必读内容** — 企业不太想让你明白的东西，都藏在附注里（→ [[books/手把手读财报/ch6-权益变动表|第六章 附注详解]]）

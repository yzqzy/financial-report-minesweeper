# Changelog

## v1.0 — 2026-04-12

初始版本，基于唐朝《手把手教你读财报（新准则升级版）》构建完整排雷体系。

### 规则体系
- 7 层 28 条规则（Layer 0-6），覆盖利润表、现金流、资产负债表、交叉验证、非财务信号、行业风险
- 评分体系：分层权重 + 组合加分 + 四级风险等级
- Layer 0 门槛检查（审计意见 + 年报按时披露）一票否决

### 数据源
- Tushare API（主）：结构化财务数据，10+ 年历史
- 年报 PDF（辅）：附注、董事会报告、关联交易等定性分析
- 复用龟龟投资框架的 `tushare_collector.py`

### 知识库
- 8 章读书笔记（`手把手读财报/ch1-ch8`）
- 原书 PDF

### Skill
- `/minesweeper <股票代码> [年份]` 一键排雷
- 自动获取数据 → 逐条检查 → 输出风险等级报告
